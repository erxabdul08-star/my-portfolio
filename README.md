# Portfolio Website

A simple personal portfolio website for **erxabdul08-star**.

## Features

- About Me section
- Projects showcase
- Contact information
- Clean and modern design

## Getting Started

1. **Clone the repository:**
   ```
   git clone https://github.com/erxabdul08-star/portfolio.git
   ```
2. **Open `index.html` in your browser to view the site.**

## Customization

- Edit `index.html` to update your bio, projects, and contact info.
- Modify `styles.css` for colors/layout.
- Add interactive features in `script.js`.
